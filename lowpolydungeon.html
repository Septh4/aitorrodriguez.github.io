Low Poly Dungeon

Role 3D Artist & Character Designer
Tools Blender, Godot 3.6
Status Prototype (Single-player functional)

Project Overview

This was a project made with one of my collaborators. We did some small projects together and we wanted to start playing with 3D. Since he's a master coder, we tried to make a small cooperative game for two players with one dungeon map. Although the single player worked perfectly, multiplayer had a load of issues that we never fully resolved.

3D Modeling Character Design Level Design Texturing Blender Low Poly Art

Character Design

My job was to create the characters and map design. So I did some small characters inspired by Minecraft, Unturned, and similar low poly, cost-effective character designs. The goal was to create readable, expressive characters that would be quick to produce and easy to animate.

Initial character concepts — Zabyr "the poisonous", Valus "the paladin", and Skeletal enemy

Character design document with stats, abilities, and cooldowns for each playable character

For optimization purposes, all characters in the game share a single skeleton rig. This allows for efficient animation reuse and reduces memory overhead — critical for a small indie project targeting smooth multiplayer performance.

Blender setup showing the shared skeleton system — all characters, enemies, and weapons use the same armature

Level Design

Afterwards I started thinking about the dungeon layout. After creating the initial 3D concept and getting approval from my collaborator, I started making the tileset. The modular approach allowed us to quickly iterate on room layouts and create varied environments.

Early 3D concept for the dungeon environment

Character Gallery

As the modeling was easy and fast, I had a lot of free time while the architecture was being designed by my collaborator. So I remade the wizard character and created several more — expanding the roster with different classes, enemies, and NPCs.

Character Remake

Some time after the project went dormant, I went back and remade the character style. I always had that perfectionist tick thinking that the UVs of the characters at that time were shit — so I rebuilt them with cleaner topology and proper UV unwrapping.
