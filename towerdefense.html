The Project

A tower defense game inspired by the Kingdom Rush saga. I was hired by Space Hyena to handle game design, sprite artwork, and visual direction. The game features elven defenders protecting the forest frontier from orc invasions, with a tiered tower upgrade system and multiple playable zones.

My responsibilities included designing the tower progression system, creating character and enemy spritesheets, illustrating level maps, and documenting the overall game structure including UI flows, world maps, and zone progression.

⏸ Development paused. Space Hyena is currently working on a colony simulator for Steam. Project may resume in the future. spacehyena.com

Character Design

The main character, Elune, is an elven archer who guides the player through the campaign. I started with concept sketches, then created full spritesheets for animation. The visual style balances readability at small sizes with enough detail to feel premium.

Tower Development

Elven towers are built on trees, reflecting the race's deep connection with nature. The forest spirit can be used to imbue attacks with magic effects. The system features three tiers of upgrades, each changing the tower's appearance and capabilities.

Tower tier progression — Watchtower (basic arrows) → Mark of the Forest (extra damage) → Veteran Hunter (extended range)

Enemy Forces

The goblin army includes multiple unit types with distinct silhouettes and behaviors. The spritesheet covers idle, walk, attack, and death animations for each variant, plus elite and armored versions for later levels.

Complete goblin army spritesheet — grunts, armored, berserkers, and death animations
